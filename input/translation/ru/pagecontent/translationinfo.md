# Обратная связь коннектафона

Мы приветствуем вашу обратную связь во время коннектафона! Поделитесь своим опытом, проблемами и успехами.

## Как оставить обратную связь

Используйте наш [Документ обратной связи коннектафона](https://docs.google.com/document/d/1PdQ8zBI9xkISP3tAqIK8-TGMql3kVVZ4UNoHVYqCy4Y/edit?usp=sharing) для:

- Сообщения о проблемах с профилями и руководствами по внедрению
- Обратной связи по функциональности API
- Документирования ваших успехов со скриншотами и записями
- Вопросов и запросов на разъяснения

Ваша обратная связь помогает нам улучшить Цифровую платформу здравоохранения Узбекистана.
